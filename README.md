Butterfly
=========

> A Scala-based mini text editor

The name refers to this [xkcd](http://xkcd.com/378/) 

Build & Launch
--------------

This project is build with Maven :

```mvn clean package```

It will build a ```.jar``` inside the ```target.``` directory

Then to launch :

```scala target/butterfly-{version}.jar```

Test
----

To run tests

```mvn test```
